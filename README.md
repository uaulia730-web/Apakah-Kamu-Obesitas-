# Apakah-Kamu-Obesitas-
Obesitas adalah penumpukan lemak tubuh berlebih yang dapat meningkatkan risiko penyakit. OBE-BOOST hadir sebagai web prediksi risiko obesitas yang mudah digunakan. Sistem ini mengklasifikasikan tingkat obesitas menggunakan Ensemble LightGBM-CatBoost dan SMOTE untuk menangani data tidak seimbang.
